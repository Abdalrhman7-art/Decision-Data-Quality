# Decision-Data-Quality
A data quality and decision-oriented analysis of wholesale and retail market price data in Syria.

## Project Overview
This project examines the quality and reliability of wholesale and retail market price data used for economic and humanitarian decision-making in Syria.
The focus is on data quality assessment, exploratory data analysis, and decision-oriented insights rather than predictive modeling.

## Data Sources
- Wholesale price data
- Retail price data

(Data is processed locally and only analytical outputs are shared.)

## Project Status
- Exploratory Data Analysis (EDA): In progress
- Data Quality Assessment: In progress

## Tools Used
- Microsoft Excel (Power Query, Pivot Tables)
- GitHub (documentation & versioning)

## Notes
This project is developed as part of a learning portfolio for MSc Data Science preparation.
